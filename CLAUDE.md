# Project Rules

## Privacy — No Linkable Blockchain Data

Do NOT include any information in tracked files (code, docs, configs) that could link this repository to a specific wallet or trading activity. This includes:

- Wallet addresses (public keys)
- Transaction hashes
- Order IDs
- Position or fill details with exact amounts that could be matched on-chain

**Allowed exceptions** — public protocol constants required for code to function:
- Published smart contract addresses (e.g. Polymarket exchange contracts, USDC token address)
- These are the same for all users and are publicly documented by the protocol
